AIM ARINC 429

The AIM ARINC 429 Custom Device configures AIM GmbH hardware designed for avionics communications using the ARINC 429 protocol. This addon configures only one AIM device. You may use multiple AIM ARINC 429 addons in your VeriStand project.

What do you want to do?

Depending on your goal, complete the following task.

Goal Task
View/Edit Hardware Configuration Configure the hardware device on this page.
Import Parameter Configuration Import the parameter configuration file in the Configuration Files section.
View Hardware Channel, Label, and Parameter Configurations Select a hardware channel, a label, or a named parameter in the system tree.

Device Configuration

This device has the following configuration settings:

  • Name—Specifies the custom device name.
  • Description—Documents this instance of the custom device.
  • PXI Slot Number—Specifies the PXI Slot Number of the hardware device.
  • Coupling Mode—Specifies the Coupling Mode of the hardware device:
    • External—All channels will send/receive physically to/from bus.
    • Dual—All channels will send/receive physically to/from bus and the send data is also internally looped to its peer channel.
    • Internal—All channels are physically disconnected from bus. Sent data is internally looped to the peer channel.
